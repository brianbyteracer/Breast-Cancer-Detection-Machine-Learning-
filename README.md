Breast Cancer Detection using Neural Network

A Machine Learning project that predicts whether a breast tumor is Benign or Malignant using a Neural Network model.
The model was trained on a dataset containing 550 rows and 30 medical features, achieving an accuracy of 93%.

Aim

The main aim of this project is to develop a Breast Cancer Detection System using a Neural Network model that can accurately predict whether a tumor is Benign or Malignant based on medical diagnostic features. This project focuses on applying Machine Learning and Deep Learning techniques to assist in early cancer detection and improve prediction accuracy in healthcare applications.
